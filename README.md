# 墨韵 · 数字水墨诗画实验室

> 落笔生花，墨染乾坤。在数字世界里，写一首东方的诗。

以代码为笔，以屏幕为宣纸。融合 SVG 水墨场景动画、Canvas 墨迹扩散模拟、题诗作画与琴韵音景的沉浸式数字水墨互动平台。

## 功能

- 🖼 **题诗作画** — 选择画心模板，题诗落款，一键导出
- 🎨 **五套主题色** — 墨韵/青花/金碧/朱砂/雪景
- 🖌 **墨韵万象** — 水墨画卷动态背景（山峦·竹林·仙鹤·锦鲤·墨滴·飞瓣）
- 💬 **素笺留墨** — 毛笔风留言墙
- 🎵 **琴韵音景** — Web Audio 古琴泛音

## 部署

纯静态单文件，开箱即用：

```bash
git clone https://github.com/oldking-yes/ink-studio.git
cd ink-studio
# 直接在浏览器打开 index.html
```

或访问 GitHub Pages：`https://oldking-yes.github.io/ink-studio/`

## 技术栈

- 纯 SVG + CSS 动画（水墨场景引擎）
- Canvas 2D（题诗作画）
- Web Audio API（琴韵音景）
- 零构建，零依赖
